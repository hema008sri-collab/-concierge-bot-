# Multilingual Translator App

## Overview
A Python-based translator that detects language, preserves names/places, and translates text into multiple languages. Designed for tourism and hackathon use.

## Features
- Language detection
- Entity recognition (names, places, travel terms)
- Accurate translation with entity protection
- Simple CLI interface
